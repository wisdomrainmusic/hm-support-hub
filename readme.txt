HM Support Hub merges support tickets from client sites into a single panel for easier management.

== Description ==
HM Support Hub provides the foundation for a centralized support workflow. It registers a core loader and admin asset pipeline so you can extend it with custom dashboards, reporting, and automation specific to your team.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/hm-support-hub` directory.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Configure your support workflow by extending the plugin classes.
