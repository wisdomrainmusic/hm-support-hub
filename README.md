# HM Support Hub

HM Support Hub is a WordPress plugin that consolidates customer support requests from multiple client sites into a single control panel.

## Features
- Centralized loader that bootstraps plugin functionality.
- Admin-facing stylesheet hook ready for future dashboard elements.
- Translation-ready structure.

## Installation
1. Upload the plugin files to the `/wp-content/plugins/hm-support-hub` directory.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Customize the plugin as needed to fit your support workflow.

## Development
- Core plugin constants and hooks are defined in `hm-support-hub.php`.
- Extend functionality by adding new classes under `includes/` and enqueueing assets from `assets/`.
